### System And Database Triggers
আমরা চাইলে Oracle সিস্টেমের সিস্টেম রিলেটেড বিষয়গুলো সেটার সিস্টেম আর ডাটাবেজ রিলেটেড ট্রিগার দিয়ে দেখতে পারি। 

### Activities Covered By System and Database Triggers
* Server start,
* Server stop,
* errors from server,
* user log on,
* user log off.
